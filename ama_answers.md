
# AMA – 23/02/2026

---

### Q1) Explain loops

Loops are used to repeat a block of code multiple times.
Python has `for` loop and `while` loop.
They help avoid writing repeated code.

---

### Q2) What is indentation?

Indentation means giving space at the beginning of a line.
Python uses indentation to define blocks of code.
Without proper indentation, Python gives an error.

---

### Q3) Methods in OOPS

Methods are functions defined inside a class.
They define the behavior of an object.
There are instance methods, class methods, and static methods.

---

### Q4) Mutable and Immutable

Mutable objects can be changed (like list, dict).
Immutable objects cannot be changed (like int, string, tuple).
Once created, immutable values stay the same.

---

### Q5) `ls` command

`ls` is a Linux command.
It shows files and folders in the current directory.
Used in terminal to list contents.

---

### Q6) What is Git?

Git is a version control system.
It tracks changes in code.
It helps multiple developers work together.

---

### Q7) What is venv?

A virtual environment (venv) creates an isolated Python environment.
Each project can have its own packages.
It prevents dependency conflicts.

---

### Q8) Explain `init` Constructor

`__init__` is a special method in a Python class.  
It runs automatically when an object is created.  
It is used to initialize (assign) the object’s variables.

---

### Q9) `pass` usage

`pass` is a placeholder statement.
It does nothing but avoids syntax error.
Used when a block is required but no code is needed yet.

---

### Q10) What is SOLID?

SOLID is 5 design principles in OOPS.
It helps write clean, maintainable code.
It improves scalability and flexibility.

---

### Q11) Creating a venv

Use command: `python -m venv venv`
Then activate it using `source venv/bin/activate` (Linux/Mac).
It creates an isolated Python environment.

---

### Q12) Difference between Abstraction and Encapsulation

Abstraction hides implementation details.
Encapsulation hides data and protects it.
Abstraction focuses on behavior, encapsulation focuses on safety.

---

### Q13) Object Creation

Object is created from a class.
Example: `obj = ClassName()`
It allocates memory and calls the constructor.

---

### Q14) `append()` and `extend()`

`append()` adds one element to a list.
`extend()` adds multiple elements from another iterable.
`append()` adds as a single item, `extend()` merges items.

---

### Q15) Difference between `is` and `==`

`==` checks value equality.
`is` checks memory location (identity).
`is` returns True only if both variables point to same object.

---

### Q16) Types of Inheritance

Single inheritance
Multiple inheritance
Multilevel inheritance
Hierarchical inheritance
Hybrid inheritance

These define how classes inherit from other classes.
